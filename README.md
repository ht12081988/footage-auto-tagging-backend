# Footage Auto Tagging Backend

This is the FastAPI Python backend for the Footage Auto Tagging application (Sentinel AI / Vigilant.ai). It runs inference using YOLOv8, RT-DETR, EasyOCR, and SigLIP embeddings, connecting to the remote Prisma PostgreSQL database.

## Deployment Info
* **SDK**: Docker
* **Port**: 7860
